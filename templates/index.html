Это тестовый проект стартапа Pavel & Co.

Описание

Проект построен в образовательных целях. Поиск выполняется через API сайта HH.ru По итогу система вытянет наиболее актуальные навыки для какой либо вакансии и сохранит в файл формата xlsx и txt.
В конце выполнения программы формируется логирование ошибок и результата.
